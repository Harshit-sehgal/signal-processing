7. STAGE 1 — DECOMPOSITION
==================================================

Stage 1 is responsible for adaptive preprocessing and CEEMDAN decomposition.

Use the algorithm name accurately.

If the code uses PyEMD `CEEMDAN`, call it CEEMDAN in code, reports and figures.

Do not label it ICEEMDAN unless a genuine and verified ICEEMDAN implementation is added.

--------------------------------------------------
7.1 Stage 1 processing requirements
--------------------------------------------------

For each recording:

1. Validate the input.
2. Select one controlled source segment.
3. Test configured candidate filter cutoffs on that same segment.
4. Select the best cutoff using a meaningful decomposition-quality objective.
5. Preprocess the full signal using the selected cutoff.
6. Extract the chosen segment.
7. Run final CEEMDAN decomposition.
8. Separate physical IMFs and residual explicitly.
9. Preserve reconstruction traceability.
10. Record all parameters and random seeds.

The final CEEMDAN output must make clear whether the final row is:

- A residual produced by the implementation, or
- Another IMF

Do not blindly assume the final row is a residual without verifying the library behaviour.

7.2 Stage 1 quantitative metrics
--------------------------------------------------

Calculate and save:

- Number of IMFs

- Signed orthogonality index
- Mean adjacent IMF correlation
- Maximum adjacent IMF correlation

- Per-IMF energy percentage
- Per-IMF centre frequency
- Per-IMF bandwidth
- Per-IMF spectral entropy
- Seed-stability metrics
- Cutoff-search metrics
- CEEMDAN runtime

Frequency ordering must be mathematically correct:

Ideal EMD ordering means centre frequency generally decreases as IMF index increases.

Test this with genuinely descending synthetic IMF frequencies.

Seed stability must not rely only on reconstruction NRMSE because all valid CEEMDAN decompositions can reconstruct the source nearly perfectly.

Seed stability should compare:

- IMF centre frequencies
- IMF energy distributions
- Matched-IMF correlations
- Spectral-overlap variation
- IMF-count variation
